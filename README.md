# qq群聊机器人
基于moqqx机器人框架提供的HTTPAPI插件实现机器人功能

框架地址：https://moqqx.cn/

插件地址：https://moqqx.cn/pages/198c35/#%E5%9F%BA%E7%A1%80%E9%85%8D%E7%BD%AE

背景是起点小说《这游戏也太真实了》的运营官想每天能在群里通知小说有更新

我就想到直接用qq机器人根据更新模板自动生成每日更新提醒，并且群发到所有群
